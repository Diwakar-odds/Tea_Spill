# APK Downloads

Place your latest Android APK file here using this exact filename:

- `TeaSpill-latest.apk`

After deployment, users can download it directly from:

- `/downloads/TeaSpill-latest.apk`

Quick publish flow:

1. Build APK (debug or signed release).
2. Copy APK to this folder and rename to `TeaSpill-latest.apk`.
3. Deploy website.
