# APK Downloads

Place your latest Android APK file here using this exact filename:

- `TeaSpill-latest.apk`

For smaller download size, use the signed `arm64-v8a` release APK.

After deployment, users can download it directly from:

- `/downloads/TeaSpill-latest.apk`

Quick publish flow:

1. Build signed release APKs.
2. Pick the `arm64-v8a` APK.
3. Copy it to this folder and rename to `TeaSpill-latest.apk`.
4. Deploy website.
